# AniCard-Collector

This is a rng based card collecting game. It has different anime series and cards from each series.

HOW TO USE IT

1. Look at the top left of the screen to view your "Coins", you start with 25 for now.

2. Click on the "Bleach Packs" and open the "Agent Of The Shinigami" pack.

3. When opened, click on "Collection" and the "Bleach" section. Then select a card to start making
coins.

4. Let your coins save up, buy more packs, and complete sets.

Each set has its own unlock condition you have to meet before unlocking more packs.

God Bless and Have Fun!
